## ChatMSG
> 공식적인 메세지 작성에 어려움을 겪었던 경험 기반으로 ChatGPT 기반 메세지 생성 앱 개발
### Tech
- Architecture : `CleanSwift` 
- Library : `SnapKit`, `OpenAISwift`
- Persistence : `Coredata`
